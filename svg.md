
### Path

属性d中可用命令：

- M = moveto
- L = lineto
- H = horizontal lineto 横向
- V = vertical lineto 竖向
- C = curveto 曲线
- S = smooth curveto 光滑曲线
- Q = quadratic Bezier curve 二次贝塞尔曲线
- T = smooth quadratic Bezier curve 
- A = elliptical Arc 椭圆弧
- Z = closepath 闭合
